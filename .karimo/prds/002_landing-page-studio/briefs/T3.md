## Implementation Notes

### TypeScript Shape

```typescript
// skills/landing-page-studio/assets/direction-taxonomy.ts

export interface DirectionDefinition {
  /** Machine key — used in BrandSpec.direction */
  key: string;
  /** Human-readable label for UI display */
  label: string;
  /** 1-2 sentence description of the aesthetic */
  description: string;
  /**
   * Vocabulary injected into generation prompts.
   * At least 4 strings — adjectives, nouns, style descriptors.
   */
  promptVocabulary: string[];
  /**
   * CSS design hints for implementers.
   * At least 3 strings — property tendencies, not actual CSS.
   * Example: "border-radius: 0", "font-family: monospace"
   */
  cssHints: string[];
}

export const DIRECTION_TAXONOMY: readonly DirectionDefinition[] = [
  {
    key: "brutalist-grid",
    label: "Brutalist Grid",
    description:
      "Raw, uncompromising aesthetic with high contrast and visible structure. " +
      "No ornament — function exposed as form.",
    promptVocabulary: ["raw", "unpolished", "high-contrast", "structural", "monospace", "grid-exposed"],
    cssHints: ["border-radius: 0", "font-family: monospace", "border: 3px solid black", "background: white or black only"],
  },
  // ... 14 more entries
] as const;

/** Union of all valid direction keys */
export type DirectionKey = typeof DIRECTION_TAXONOMY[number]["key"];

// Convenience set for O(1) lookup
const DIRECTION_KEY_SET = new Set(DIRECTION_TAXONOMY.map((d) => d.key));

export function isValidDirection(key: string): key is DirectionKey {
  return DIRECTION_KEY_SET.has(key);
}
```

## Gotchas

**Exactly 15 directions — no more, no fewer.**
The orchestrator in T4 will validate `BrandSpec.direction` against this list. If the count is wrong, downstream tasks will break. Count explicitly before committing.

**`as const` is required on the array.**
Without `as const`, TypeScript cannot infer the union type `typeof DIRECTION_TAXONOMY[number]["key"]`. The `DirectionKey` type derivation depends on it.

**Keys must be kebab-case, lowercase.**
The keys are user-facing in `BrandSpec.direction`. Use `kebab-case` only (e.g. `"clean-minimal"`, not `"cleanMinimal"` or `"clean_minimal"`).

**`cssHints` are descriptive, not executable.**
They describe CSS tendencies for a human implementer or LLM to interpret — not actual CSS to inject verbatim. Keep them as strings like `"border-radius: 0"` or `"heavily weighted typography"`, not CSS classes or selectors.
