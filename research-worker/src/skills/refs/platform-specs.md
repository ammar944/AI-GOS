# Platform Creative Specifications

> Source: Platform official documentation and advertising guidelines (2024–2025).
> Verify specs at platform-specific help centers before campaign launch, as platforms update
> requirements periodically.

## Google Ads

### Search Ads (Responsive Search Ads — RSA)

| Element        | Specification                                     |
|----------------|---------------------------------------------------|
| Headlines      | Up to 15 headlines; 30 characters each            |
| Descriptions   | Up to 4 descriptions; 90 characters each          |
| Final URL      | Must match display URL domain                     |
| Display URL    | 15 characters per path field (2 path fields)      |
| Ad strength    | Aim for "Excellent" — 8–10 unique headlines minimum|

**Pinning note**: Pin headlines/descriptions sparingly — over-pinning reduces RSA performance. Pin only legal disclaimers or brand-critical terms.

### Display Ads (Responsive Display Ads)

| Asset Type        | Requirements                                               |
|-------------------|------------------------------------------------------------|
| Short headline    | Up to 5; 30 characters each                               |
| Long headline     | Up to 5; 90 characters each                               |
| Description       | Up to 5; 90 characters each                               |
| Business name     | 25 characters                                             |
| Logo (landscape)  | 1200×300 px minimum; max 5120 KB; ratio 4:1               |
| Logo (square)     | 128×128 px minimum; max 5120 KB; ratio 1:1                |
| Image (landscape) | 1200×628 px minimum; max 5120 KB; ratio 1.91:1            |
| Image (square)    | 1200×1200 px minimum; max 5120 KB; ratio 1:1              |
| Image (portrait)  | 900×1600 px minimum; max 5120 KB; ratio 9:16              |
| Video (landscape) | 1280×720 px; H.264; 30fps; min 5 sec                      |

### Performance Max Ads

Combines Search + Display + Shopping + YouTube + Discovery. Supply all available asset types for best coverage. Same specs as RSA + Display + Video above apply.

### YouTube — Video Ad Formats

| Format               | Specs                                                                          |
|----------------------|--------------------------------------------------------------------------------|
| Skippable In-Stream  | 12 sec – 6 min; skip after 5 sec; MP4/MOV; 1080p recommended; min 480p       |
| Non-Skippable        | Max 15–20 sec (varies by region); no skip; high CPM                            |
| Bumper Ad            | Max 6 sec; no skip; pure awareness play                                        |
| Video Discovery      | 1280×720 or 640×480; shown in search results; user initiates                   |
| Masthead             | 1920×1080; reserved buy only; 30-day CPD pricing                               |

**Safe zones for YouTube**: Keep branding and key messaging in the center 80% of frame; avoid bottom 20% (overlaid by ad controls) and top 10% (overlaid by info cards).

### Shopping Ads (Product Listing Ads)

| Element            | Requirements                                              |
|--------------------|-----------------------------------------------------------|
| Product image      | 100×100 px minimum; white background preferred; 1MB max  |
| Title              | 150 characters; first 70 shown in most placements         |
| Description        | 5000 characters; first 180 shown                          |
| Price              | Must match landing page; currency must be specified       |
| GTIN / MPN         | Strongly recommended for all products                     |

| Placement            | Recommended Size     | Minimum Size       | Ratio        | File Max |
|----------------------|----------------------|--------------------|--------------|----------|
| Facebook Feed        | 1200×628 px          | 600×314 px         | 1.91:1       | 30 MB    |
| Instagram Feed       | 1080×1080 px         | 500×500 px         | 1:1          | 30 MB    |

|----------------------|----------------------|--------------|-----------------|----------|

**Safe zone**: Top 14% and bottom 20% of frame are covered by UI elements. Keep all critical text/CTAs in the center 66% of the vertical frame.

## Universal Creative Best Practices

### Video Ads (All Platforms)

- Hook in first **2–3 seconds** — primary retention drop-off point
- Include **closed captions** — 70–85% of video is watched muted across all platforms
- Brand logo visible within **first 3 seconds** (brand awareness) or **after value delivery** (performance)
- **CTA placement**: Last 5 seconds AND as overlay text throughout for conversion campaigns
- Native aspect ratio: 9:16 vertical for Stories/Reels/TikTok; 1:1 square for Feed; 16:9 landscape for YouTube
- Avoid music with licensing issues — use platform-provided music libraries or royalty-free sources

### Image Ads (All Platforms)

- High contrast between subject and background
- Single focal point — avoid cluttered compositions
- Test: stock-photo style vs. lo-fi/UGC aesthetic (UGC often outperforms in cold prospecting)
- Faces perform well on Meta; product-focused images perform well on Shopping
- A/B test: text overlay present vs. absent
