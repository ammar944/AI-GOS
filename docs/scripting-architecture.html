AIGOS Ad Scripting Engine

Architecture Overview — Internal Technical Reference
1 High-Level Flow
▶
Frontend — Profile > Scripts Tab
User clicks "Generate New Batch". Frontend selects the most complete research session (highest section count, not just most recent).
▼
API — POST /api/scripts/generate
Fetches profile (company, style refs, proof points) + research session.
Filters to completed sections only (status === 'complete').
Runs trimResearchForScripts() to extract priority data (~8k tokens).
Creates script_packs row (status: generating).
Awaits worker dispatch with 10s timeout. Returns error immediately if worker is down.
▼
Worker — POST /api/scripts (Railway)
Responds 202 immediately. Runs async:
Loops 5 awareness levels × (Pass 1 draft + Pass 2 humanize).
Progressive Supabase writes after each level (3, 6, 9, 12, 15 scripts).
After all levels: Pass 3 batch diversity validator.
Final write: status = complete + diversity_score + diversity_flags.
▼
Frontend — Realtime Polling
useScriptPackRealtime polls GET /api/scripts/{packId} every 2 seconds.
Scripts appear progressively as each awareness level completes.
Stale detection: if generating with 0 scripts for >5 min, shows timeout error.
On complete: loads diversity flags + score into the UI.
2 3-Pass Generation Pipeline
▶

Each awareness level runs Pass 1 + Pass 2 sequentially. Pass 3 runs once after all levels.

Unaware Long copy, story-driven
Problem Name the pain
Solution Position vs alternatives
Product Handle objections
Most Aware Offer-forward, short

Pass 1: Draft Sonnet 4.6

Generate 3 scripts per level using:

  • Trimmed research context (~8k tokens)
  • ICP monologue triggers (Collier framework)
  • Proof points (or "NO PROOF" directive)
  • Style references from winning ads
  • Platform specs (from platform-specs.md)
  • Ad copy templates (from ad-copy-templates.md)
  • 6 DR frameworks (Schwartz, Hopkins, Ogilvy, Caples, Sugarman, Collier)
  • Angle dedup from previous levels
  • Kill list (17 phrases, 22 words, 8 structures)

Pass 2: Humanize 43 Checks

Ruthless AI-copy editor audit:

  • 7 Content checks (inflation, name-dropping, vague attributions...)
  • 7 Language checks (AI vocab tiers, copula, template phrases...)
  • 11 Structure checks (em dash audit, sentence rhythm, uniform paragraphs...)
  • 13 Communication checks (chatbot closers, "let's" constructions...)
  • 5 Ad-specific checks (weak hooks, feature-first, missing specificity...)

Then a second pass catches survivors from the rewrite itself.

Pass 3: Diversity Batch Review

After all 15 scripts, AI creative director reviews:

  • Angle diversity (duplicate angle pairs)
  • Hook diversity (similar language)
  • Format coverage (video, static, email)
  • Platform coverage (Meta, Google, LinkedIn)
  • CTA variety

Outputs: diversity score (0-10) + flags shown in UI.

Total per batch: 10 AI calls (5 levels × 2 passes) + 1 diversity call = 11 Sonnet 4.6 calls. ~3 min timeout per call. Progressive writes after each level.

4 Assets Pipeline (Proof Points + Style Refs)
▶
User Input — Assets Tab
Style References: Paste winning ads, scripts, or hooks. Name + source URL + full copy text.
Proof Points: Type (Case Study / Testimonial / Metric / Credential) + headline + detail + client name + verified toggle.
▼
Persistence — PUT /api/profiles/{id}/style-references
Saves to business_profiles.style_references and business_profiles.proof_points (both JSONB). Same endpoint handles both.
▼
Generation — Injected into Pass 1 Prompt
Style refs: "STYLE REFERENCES" section — AI must match voice, cadence, rhythm.
Proof points: "AVAILABLE PROOF" section with type, headline, detail, client name.
No proof? "NO VERIFIED PROOF AVAILABLE. Do not fabricate. Flag claims needing proof in flaggedClaims."
5 Quality Controls
▶

43-Check Audit (Pass 2)

7 content + 7 language + 11 structure + 13 communication + 5 ad-specific. Every script gets two sub-passes: fix, then catch survivors from the rewrite.

Em Dash Kill

Target: zero em dashes per batch. The single most visible AI fingerprint in ad copy.

Sentence Rhythm

Every script must have at least one sentence under 5 words AND one over 25 words. Flat rhythm = robotic.

Angle Dedup

Tracks used angles + hooks across levels. Each new level must choose different angles. Min 2/3 unique per level.

Proof Grounding

Every claim must cite a research section. Unverifiable claims go into flaggedClaims. No fabrication.

Diversity Score (Pass 3)

AI creative director rates 0-10 for angle/hook/format/platform/CTA variety. Flags shown as "Batch Review Notes".

Kill List

17 banned phrases, 22 word replacements (leverage→use, streamline→simplify...), 8 structural bans.

AI Vocabulary Tiers

Tier 1: Always replace (additionally, crucial, landscape, delve...). Tier 2: Replace if >1/script. Tier 3: Replace if 3+ in one piece.
