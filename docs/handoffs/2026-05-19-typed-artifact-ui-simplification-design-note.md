# Typed Artifact UI Simplification Design Note

Prose leads every section in a narrow reading column; supporting evidence only appears after the argument has landed. Data tables are quiet annotations, not structural headlines, and they should lose card chrome unless comparison is unavoidable. Quote callouts are typographic moments with a left rule and attribution, not boxed widgets. Positioning axes should render as visual axes, not as repeated cards. The preview should feel like reading a thoughtful agent memo inside a Claude-style output pane, not operating a dashboard.
