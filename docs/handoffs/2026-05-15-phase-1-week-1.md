| Dimension | Opus baseline | Sonnet PASS | Sonnet FAIL |
|---|---|---|---|
| `streamObject` completes | yes | yes | timeout / schema reject |
| 5 sub-sections present | yes | yes | any missing |
| Post-validate ok | first emission | within 1 retry | unfixable after 1 retry |
| Walltime | ~372s | ≤ 600s | > 900s |
| Cost (input + output tokens) | ~$2 | ≤ $1.20 | > $2 |

| Persona count | ≥ 5 named real people | ≥ 5 named real people | < 5 |

- **PASS** = all 7 dimensions in the "Sonnet PASS" column
- **PARTIAL** = 5-6 / 7
- **FAIL** = ≤ 4 / 7
