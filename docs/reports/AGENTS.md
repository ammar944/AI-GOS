# AGENTS.md - docs/reports

## Purpose

- Owns evidence reports, QA reports, review findings, and report assets.

## Ownership

- Reports own observed facts from a specific run, review, or investigation.
- `assets/` owns supporting images or artifacts for reports.

## Local Contracts

- Separate facts, judgments, assumptions, and recommendations.
- Include exact commands, dates, paths, run IDs, URLs, screenshots, DB queries, or source evidence when relevant.
- Do not turn a failed or partial proof into a green status.

## Work Guidance

- Keep reports readable from current docs without private memory.
- Prefer concise findings with evidence over broad narrative.

## Verification

- Re-check referenced evidence when updating a report, or state clearly when evidence is historical.
